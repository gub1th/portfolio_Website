Winter 2021

For my winter break, I decided that I didn't want to be entirely unproductive. So, without much experience with web development, I decided to make my own portfolio website for fun. It would be great to document everything that's been going on while gaining more knowledge in webdev-oriented languages like HTML, CSS and Javascript.

I used a template for the design (credit to CodingNepal) but have been customizing everything to my liking, including making these new pages for the projects. This website is still under development, particularly in terms of information.
